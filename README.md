# AR Geolocalización - Casa Abierta UIDE

Este proyecto utiliza **AR.js** y **A-Frame** para mostrar objetos 3D en coordenadas GPS del mundo real.

## Cómo usarlo
1.  Abre index.html.
2.  Busca la etiqueta <a-entity> y cambia los valores de latitude y longitude por los del lugar donde quieras que aparezca el objeto.
3.  Sube este archivo a un servidor con **HTTPS** (GitHub Pages es ideal).
4.  Abre el enlace en tu móvil, otorga permisos de cámara y GPS, y busca el objeto en el horizonte.

## Requisitos
*   Servidor con HTTPS.
*   Dispositivo móvil con GPS y Brújula.
